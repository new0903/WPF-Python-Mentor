Важное примечание по поводу input и print
Каждый элемент в CorrectAnswer представляет собой каждую строку консоли
если код пользователя будет например таким:
name=input("enter your name")
print("your name = "+name)
word=input("enter any word")
print("your word = "+word)
то correctAnswer должен выглядеть вот так

"enter your name your name = <здесь будет содержимое name>",
"enter any word your word = <здесь будет содержимое word>"

это обработанная строка с помощью split и trim
не обработанная строка будет выглядеть примерно так
"enter your name your name = <name>\n\renter any word your word = <word>\n\r"
так что учитывай этот вариант в задачах с кодом

Данные для input записываем строками в option вот так ["тест", "input что то там"]

эти данные будут вставлятся в input

Скрипт будет работать в течении 60 секунд и если задача так и не будет выполнена приложение автоматически закроет процесс

ВАЖНО!!! Сохраняйте структуру вопроса иначе вы не добавите вопросы в раздел
{
  "Name": "Название задачи",
  "Description": [
    { "Header": "Местоположение / Сцена" },
    { "Simple": "Нарратив в стиле Ведьмака: диалог, описание, объяснение темы" },
    ...
  ],
  "Option": ["данные для input() или варианты ответа"],
  "CorrectAnswer": ["ожидаемый вывод программы или правильные варианты"],
  "SupportHint": "Краткая подсказка, если не нужна подсказка просто добавьте пустое поле ''",
  "SupportAnswer": "Подробное решение или объяснение (можно с кодом), если не нужна подсказка просто добавьте пустое поле ''",

}

Если в задаче надо писать код ОБЯЗАТЕЛЬНО ПРОСТАВИТЬ "IsCode": true
